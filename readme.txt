== Description ==

This plugin just replace the font of Wordpress with Uighur font, and you can direct type Uighur characters on backend (write posts) and front-end (write comments). You can get obvious effects in the Uighur version of wordpress. You can select several kinds of fonts on your profile page， different users can choose his own font.

= Features =

* Several kinds of fonts are optional
* Direct type Uighur characters (using ug_vk.js : http://finalfantasia.github.io/ug_vk/)

= Requirements =

* WordPress 3.6 or higher.
    	
== Installation ==

* Extract the zip file and just drop the contents in the <code>wp-content/plugins/</code> directory of your WordPress installation (or install it directly from your dashboard) and then activate the Plugin from Plugins page.
